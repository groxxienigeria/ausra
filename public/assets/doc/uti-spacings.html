Spacings

Folio includes a wide range of shorthand responsive margin and padding utility classes to modify an elementâ€™s appearance.

How It works

Assign responsive-friendly margin or padding values to an element or a subset of its sides with shorthand classes. Includes support for individual properties, all properties, and vertical and horizontal properties. Classes are built from a default Sass map ranging from .25rem to 15rem.

Notation

Spacing utilities that apply to all breakpoints, from xs to xl, have no breakpoint abbreviation in them. This is because those classes are applied from min-width: 0 and up, and thus are not bound by a media query. The remaining breakpoints, however, do include a breakpoint abbreviation.

The classes are named using the format {property}{sides}-{size} for xs and {property}{sides}-{breakpoint}-{size} for sm, md, lg, and xl

Where property is one of:

  • gr-m - for classes that set margin

  • gr-p - for classes that set paddings

Where sides is one of:

  • t - for classes that set margin-top or padding-top

  • b - for classes that set margin-bottom or padding-bottom

  • l - for classes that set margin-left or padding-left

  • r - for classes that set margin-right or padding-right

  • x - for classes that set both *-left and *-right

  • y - for classes that set both *-top and *-bottom

  • blank - for classes that set a margin or padding on all 4 sides of the element

Where size is one of:

  • 0 - for classes that eliminate the margin or padding by setting it to 0

  • 1 - (by default) for classes that set the margin or padding to $spacer * .375

  • 2 - (by default) for classes that set the margin or padding to $spacer * .5

  • 3 - (by default) for classes that set the margin or padding to $spacer * .75

  • 4 - (by default) for classes that set the margin or padding to $spacer,

  • 5 - (by default) for classes that set the margin or padding to $spacer * 1.25

  • 6 - (by default) for classes that set the margin or padding to $spacer * 1.5625

  • 7 - (by default) for classes that set the margin or padding to $spacer * 1.875

  • 8 - (by default) for classes that set the margin or padding to $spacer * 2.5

  • 9 - (by default) for classes that set the margin or padding to $spacer * 2.8125

  • 10 - (by default) for classes that set the margin or padding to $spacer * 3.125

  • 11 - (by default) for classes that set the margin or padding to $spacer * 3.75

  • 12 - (by default) for classes that set the margin or padding to $spacer * 4.375

  • 13 - (by default) for classes that set the margin or padding to $spacer * 5

  • 14 - (by default) for classes that set the margin or padding to $spacer * 5.625

  • 15 - (by default) for classes that set the margin or padding to $spacer * 6.25

  • 16 - (by default) for classes that set the margin or padding to $spacer * 7.5

  • 17 - (by default) for classes that set the margin or padding to $spacer * 9.375

  • 18 - (by default) for classes that set the margin or padding to $spacer * 12.5

  • 19 - (by default) for classes that set the margin or padding to $spacer * 15

  • auto - for classes that set the margin to auto
